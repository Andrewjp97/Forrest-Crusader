    When Will Forrest Crusader Be Released?

    Good Question, and I wish I had a good answer. Apple recently released a new programming language called Swift, and It may take a while to become comfortable programming in the new language and migrate our existing codebase from Objective-C to Swift. Along with the new language, there are still tons of graphics that need to be created, animations to be layed out, levels to design, and code to write. Taking into account all these factors, there really is no way to say when we will be done right now, but stay tuned for information and updates.

  • 02

    How much will this cost?

    $1,000,000. Just Kidding. Forrest Crusader will be available on the app store as a free download. We want to keep the game free and open because we want everyone to enjoy the game and see what is possible using Spritekit. If you would like to make a donation, please use the contact form to let us know.

  • 03

    Will Forrest Crusader Be Available on Android or Windows Phone?

    Sorry, but Forrest Crusader will be available only on iOS. We have many reasons for this; fragmentation of the android platfrom, a number of screen sizes and resolutions too large to count, and a completely different programming language and syntax. While we know this will alienate some people, we want to focus our time and effort on making a great game for one platform, rather than a mediocre one compatable with android devices.

  • 04

    Can I get Involved?

    Yes and No. If you would like to help with the graphic design elements, we would be happy to have you help. Fill out the contact form and we will get back to you about how you can help and the specifications and style of the artwork. If you would like to help program the game, unfortunately we are not looking for programers at this time.

  • 05

    What Versions of iOS Will You Support?

    Our build target will remain iOS 7 and above. The reason for this is that in iOS 7, Apple introduced a new 2D game creation framework called Spritekit which is not availble to devices running older versions of iOS. In order to fully realized the potential of the game, we want to use Spritekit for its powerful game development engine, and therefore must alienate users with older devices.

  • 06

    Why Did You Spell Forest Wrong on Everything?

    We didn't! The name is spelled Forrest Crusader because the main character is named Forrest and he is a crusader against the evil villain Decay. It's a pun. We thought it was punny. 😉
